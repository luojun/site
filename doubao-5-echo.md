# 二、为什么不同驻波态稳定性天差地别？

关键就三条，全是几何 + 物理，没有假设：

### 1. **驻波的“节点数” 决定抗扰动能力**

氢原子的每个能级 |n⟩ 都是**库仑势里的球对称驻波**
- **n=1（基态）**：0 个节点，波形最紧凑、最对称、最“刚性”
- **n=2**：1 个节点
- **n=3**：2 个节点
- 越高 n → 节点越多、波形越舒展、越“软”

**节点越多 = 驻波越容易被扰动**就像一根弦：
- 基频（无节点）最稳
- 泛音（有节点）一碰就乱

这是**几何稳定性**。

---

### 2. **高能态有大量可衰减的共振通道，基态没有**

这是稳定性差异的**真正根源**。
- **基态 n=1**
    没有任何**更低能量的驻波存在**
    → **没有任何可退的共振通道**
    → 完全封闭、无出口
    → **绝对稳定（理想情况下永不衰减）**
    
- **激发态 n≥2**
    存在**一系列更低的能级**
    → 天生自带**向下的共振衰减通道**
    → 只要一点点电磁微扰，就会触发 **ΔE = 能级差** 的共振
    → 快速衰减到低能驻波

**一句话：基态是死胡同，激发态是下坡路。**

---

### 3. **离核越远，驻波越容易和外界耦合**

高能态的电子云**离核远、范围大**
- 更容易和外界电磁场耦合
- 更容易被碰撞扰动
- 更容易触发自发辐射

这是**空间开放性导致的不稳定**。
